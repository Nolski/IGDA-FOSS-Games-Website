Amazon Lumberyard

Amazon Lumberyard is a free AAA game engine deeply integrated with AWS and Twitch – with full source.

License

Amazon Webservices Agreement, and Lumberyard Service Terms

Github

https://github.com/aws

Community Size

~28 contributors

Price

Free

Frequency of Commits

Last commit < 24 hrs ago

Non-standard Clauses

You have to use Amazon Web Services if you're going to use Lumberyard. Only certain aspects are redistributable.

Steam VR / Vive

Vive is a first-of-its kind virtual reality system developed in partnership by HTC and Valve. Designed from the ground up for room-scale VR, Vive allows true-to-life interactions and immersive experiences thanks to stunning graphics, HD haptic feedback and 360˚ motion tracking.

License

3-clause BSD-style license

Github

https://github.com/ValveSoftware/openvr

Community Size

< 150 forks

Price

Free

Frequency of Commits

2-3 per month

Non-standard Clauses

None

Unity 3D

You can create any 2D or 3D game with Unity. You can make it with ease, you can make it highly-optimized and beautiful, and you can deploy it with a click to more platforms than you have fingers and toes. What’s more, you can use Unity’s integrated services to speed up your development process, optimize your game, connect with an audience, and achieve success.

License

Custom EULA

Github

https://github.com/UnityTechnologies

Community Size

29 directly manage repos, at least several thousand forks across all 43 reposo

Price

Cannot use free version if you make over $10k a fiscal year

Frequency of Commits

Varies from "multiple daily" to "not in a few years"

Non-standard Clauses

Commerical Titles: You need to track gross revenue and pay a 5% royalty on that amount after the first $3000 per game per calendar quarter.

Unreal Engine

Unreal Engine 4 is a complete suite of game development tools made by game developers, for game developers. From 2D mobile games to console blockbusters and VR, Unreal Engine 4 gives you everything you need to start, ship, grow and stand out from the crowd.

License

Source Available with EULA

Github

Proprietary Software

Community Size

Price

Royalty charges

Frequency of Commits

Non-standard Clauses

Vulkan

Vulkan is a new generation graphics and compute API that provides high-efficiency, cross-platform access to modern GPUs used in a wide variety of devices from PCs and consoles to mobile phones and embedded platforms.

License

Various

Github

https://github.com/KhronosGroup

Community Size

~60+ contributing corporations

Price

Free

Frequency of Commits

N/A

Non-standard Clauses

Contributing to Vulkan is limited to those who are registered contributing members of the Khronos Group.
